初入Github，第一件事情做的就是建立一个博客，这样以后也能记录自己今后技术学习历程。
使用HEXO+Github建立一个博客不需要什么代码基础，基本就是跟着网上的教程一步一步走，但是也会碰到教程缺少东西的情况，要同时看多份教程，比较其中的内容。
我参考了这几个教程“hexo博客搭建时遇到的一些问题” “从零开始到搭建完整”使用Hexo+Github一步步搭建属于自己的博客（基础）
首先要安装配置环境，安装Node.js、Git环境、Hexo环境。
node.js是一个javascript的运行环境 java的运行环境不是JRE吗？这个有什么区别？
hexo是基于nodejs环境的静态博客
在blog文件夹下git bash here
安装hexo命令 npm i -g hexo
初始化hexo hexo init
在github中配置账户信息 git config –global user.name “ywf666” / git config – global user.email “ywfsdsg@gmail.com“
安装ssh，ssh-keygen -t rsa -C “ywfsdsg@gmail.com“ -f ~/.ssh/key 其中最后一个key是密钥名字，这时候就会在C：/Users/[Username]/.ssh下出现
获取SSH密钥 cd ~/.ssh cat id_key.pub
把获得的SSH复制到github里
修改_config.yml的配置信息
deploy:
type: git
repo: https://github.com/YourgithubName/YourgithubName.github.io.git
branch: master
本地测试 hexo clean \ hexo generate \ hexo server 然后就能访问heep://localhost:4000查看了
上传github hexo clean \ hexo generate \ hexo deploy 每次更新内容都要用
第一次deploy时要输入github的账号和密码
